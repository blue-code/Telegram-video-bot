# Initial Concept
Supabase를 이용한 지능형 텔레그램 미디어 보관소. 유튜브나 특정 동영상 URL을 입력하면 영상을 다운로드하여 텔레그램에 업로드하고, 저장된 미디어를 웹 대시보드에서 스트리밍, 읽기, 관리할 수 있는 통합 솔루션.

# 제품 가이드: Telegram Video Bot (TVB)

## 1. 개요
Supabase와 텔레그램을 기반으로 하는 지능형 미디어 보관 및 스트리밍 시스템입니다. 영상뿐만 아니라 일반 파일, 전자책(EPUB)까지 통합 관리하며, 텔레그램의 무제한 저장 공간을 활용하여 개인화된 미디어 라이브러리 역할을 수행합니다.

## 2. 주요 타겟 사용자
- **미디어 수집가:** 영상, 파일, 책 등을 한곳에 모아 관리하고 싶은 사용자.
- **모바일 시청자:** 모바일 기기에서 끊김 없는 영상 시청과 독서를 원하는 사용자.
- **클라우드 사용자:** 별도의 유료 스토리지 없이 텔레그램을 활용해 대용량 파일을 안전하게 보관하려는 사용자.

## 3. 핵심 기능
- **통합 미디어 다운로드:**
    - 유튜브 및 1000+ 사이트 영상 다운로드 (최고 화질, MP3 추출 지원).
    - 대용량 파일 자동 분할 업로드 (텔레그램 제한 우회).
    - **자막 자동 연동:** 동영상과 동일한 파일명의 자막(.srt, .smi) 자동 검색 및 재생 지원.
    - **자막 포맷 변환:** SMI 자막의 WebVTT 자동 변환 및 인코딩 감지 처리.
    - **내장 자막 인코딩:** 영상 최적화 시 자막을 MP4 내부에 Soft Subtitle로 병합.
- **스마트 보관 및 스트리밍:**
    - **Bin Channel:** 텔레그램 채널을 무제한 클라우드로 활용.
    - **Seamless Streaming:** 분할된 영상도 웹에서 하나처럼 실시간 스트리밍.
    - **모바일 최적화:** H.264/AAC 재인코딩으로 모바일 재생 품질 보장.
- **eBook 라이브러리 & 리더:**
    - EPUB 파일 자동 인식 및 메타데이터(표지, 작가) 추출.
    - 독서 진행 상황 자동 저장 및 테마/글꼴 커스터마이징이 가능한 웹 뷰어.
    - **TTS 오디오북:** Edge TTS를 활용한 고품질 음성 낭독 지원.
- **코믹스/만화 뷰어:**
    - ZIP/CBZ 파일 자동 인식 및 시리즈/권수 메타데이터 파싱.
    - 웹 기반 전용 뷰어 (좌우 터치, 이미지 최적화 로딩).
- **일반 파일 관리:**
    - 비디오 외 일반 파일 업로드/다운로드 및 상세 검색/필터링.
    - 다중 파일 선택 다운로드 (자동 ZIP 압축 및 텔레그램 알림).

## 4. 사용자 인터페이스 (UI)
- **텔레그램 봇:** 미디어 업로드 및 주요 작업 알림 수신 창구.
- **웹 대시보드 & 갤러리:**
    - 반응형 디자인으로 모바일/PC 최적화.
    - 통계 확인, 상세 검색, 필터링, 페이지네이션 지원.
- **전용 뷰어:**
    - 영상용 모던 HTML5 플레이어 (속도 조절, PiP, 극장 모드).
    - eBook용 ePub.js 리더 (터치 인터페이스).
