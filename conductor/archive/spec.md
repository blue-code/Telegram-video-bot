# Track: Single-Book Series Direct Read

## 1. 개요 (Overview)
현재 eBook 메뉴의 기본 화면인 '시리즈 뷰'에서, 단 1권만 포함된 시리즈(단권 도서)를 클릭할 경우 불필요하거나 작동하지 않는 상세 페이지로 이동하여 사용자가 책을 바로 읽을 수 없는 문제가 있습니다.
본 트랙은 1권짜리 시리즈의 경우, 시리즈 상세 화면을 거치지 않고 즉시 웹 리더(Reader)를 실행하여 사용자 경험을 개선하는 것을 목표로 합니다.

## 2. 기능 요구사항 (Functional Requirements)
1.  **시리즈 목록 데이터 조회 개선 (Backend):**
    *   `/books/series` 라우트에서 시리즈 목록을 조회할 때, 각 시리즈의 포함 도서 개수(`book_count`)뿐만 아니라, **첫 번째 도서의 ID**도 함께 조회되도록 쿼리를 수정합니다.
2.  **조건부 링크 렌더링 (Frontend):**
    *   `books_series.html` 템플릿에서 각 시리즈 카드를 렌더링할 때 `book_count`를 확인합니다.
    *   **Case A (book_count == 1):** 클릭 링크(`href`)를 `/books/read/{book_id}`로 설정하여 리더를 바로 엽니다.
    *   **Case B (book_count > 1):** 클릭 링크(`href`)를 기존과 동일하게 `/books/series/{series_id}`로 설정하여 시리즈 상세 화면으로 이동합니다.

## 3. 비기능 요구사항 (Non-Functional Requirements)
*   **성능:** 시리즈 목록 쿼리 시 `book_id`를 추가로 가져오는 작업이 전체 목록 로딩 속도에 눈에 띄는 저하를 주지 않아야 합니다.
*   **UI 일관성:** 링크의 동작만 변경되며, 1권짜리 시리즈와 다권 시리즈의 카드 디자인(커버, 제목 등)은 동일하게 유지합니다.

## 4. 수용 기준 (Acceptance Criteria)
*   [ ] eBook 시리즈 목록 화면에 진입한다.
*   [ ] 1권만 있는 책(시리즈)을 클릭하면 즉시 해당 책의 리더 화면이 열려야 한다.
*   [ ] 2권 이상 있는 시리즈를 클릭하면 기존대로 시리즈 상세 목록 화면이 열려야 한다.
